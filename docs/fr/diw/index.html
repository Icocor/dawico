DIW

Diseño de Interfaces Web

El módulo de diseño de interfaces web contiene parte de la formación necesaria para desempeñar la función de desarrollador de aplicaciones web.

El diseño de interfaces incluye aspectos como:

  • La planificación de la interfaz web de acuerdo con las especificaciones del diseño.
  • La creación y el mantenimiento de los elementos de la interfaz.
  • La aplicación de las guías de estilo.
  • El cumplimiento de los criterios de accesibilidad y usabilidad en el desarrollo de aplicaciones web.
